# 书籍章节拆分工具

一个 Chrome 浏览器扩展，用于将 PDF、EPUB、MOBI 等书籍文件按章节拆分为独立文档，方便在 Google NotebookLM 中进行更细致的分析。

## 功能特点

- 📚 支持多种书籍格式：PDF、EPUB、MOBI
- 🔍 智能章节识别：自动检测章节标题
- 🎯 多种识别模式：支持数字编号、中文标题、罗马数字等
- 📝 多种输出格式：纯文本、Markdown、HTML
- 💾 批量下载：一键下载所有拆分后的章节文件

## 安装方法

1. 打开 Chrome 浏览器
2. 访问 `chrome://extensions/`
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择 `book-splitter-extension` 文件夹

## 使用方法

### 方法一：通过扩展图标

1. 点击浏览器工具栏中的扩展图标
2. 点击"选择书籍文件"按钮
3. 选择要拆分的 PDF、EPUB 或 MOBI 文件
4. 配置拆分选项：
   - **自动检测章节**：启用/禁用自动章节检测
   - **章节识别模式**：选择章节标题的识别模式
   - **输出格式**：选择输出文件格式
5. 点击"开始拆分"按钮
6. 拆分完成后，点击"下载所有章节"按钮

### 方法二：在 NotebookLM 页面使用

1. 访问 Google NotebookLM 网站
2. 页面右下角会出现"📚 拆分书籍"浮动按钮
3. 点击按钮打开扩展界面

## 章节识别模式

### 自动检测
自动尝试多种模式识别章节标题

### 数字编号
识别类似以下格式的章节：
- 第1章：标题
- Chapter 1: Title
- 第 10 章：标题

### 中文标题
识别中文数字章节：
- 第一章：标题
- 第二章：标题
- 一、标题

### 罗马数字
识别罗马数字章节：
- I. 标题
- Chapter II: Title

### 自定义正则
使用自定义正则表达式匹配章节标题，例如：
- `^第[一二三四五六七八九十]+章` - 匹配中文数字章节
- `^Chapter \d+` - 匹配英文数字章节

## 输出格式

### 纯文本 (.txt)
简单的文本格式，保留原始内容

### Markdown (.md)
Markdown 格式，章节标题作为一级标题

### HTML (.html)
HTML 格式，包含基本样式，适合在浏览器中查看

## 技术说明

### 文件格式支持

- **PDF**: 使用 pdf.js 库提取文本内容
- **EPUB**: 使用 JSZip 解压并解析 OPF 文件结构
- **MOBI**: 基础文本提取（建议先转换为 EPUB 或 PDF 以获得更好效果）

### 注意事项

1. **MOBI 文件**：MOBI 格式较复杂，如果无法正确解析，建议先使用 Calibre 等工具转换为 EPUB 或 PDF
2. **大文件处理**：处理大型文件可能需要较长时间，请耐心等待
3. **章节识别**：如果自动识别不准确，可以尝试使用自定义正则表达式
4. **下载限制**：浏览器可能会限制同时下载的文件数量，如果下载失败，可以分批下载

## 常见问题

**Q: 为什么无法识别章节？**
A: 请检查章节标题格式是否符合识别模式，或尝试使用自定义正则表达式。

**Q: 处理大文件时浏览器卡死？**
A: 大文件处理需要时间，请耐心等待。如果文件过大，建议先使用其他工具进行预处理。

**Q: 下载的文件在哪里？**
A: 文件会下载到浏览器的默认下载目录。

**Q: 支持其他文件格式吗？**
A: 目前支持 PDF、EPUB、MOBI。如需支持其他格式，可以提交功能请求。
