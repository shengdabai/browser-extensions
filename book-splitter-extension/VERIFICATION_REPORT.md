## 📝 已知限制

1. **MOBI 文件支持**
   - 提供基础解析功能
   - 建议转换为 EPUB 或 PDF 使用

2. **网络依赖**
   - 首次使用需要网络连接（加载 PDF.js 和 JSZip）
   - 已添加重试机制处理网络波动

3. **大文件处理**
   - 特别大的文件可能需要较长处理时间
   - 建议文件大小 < 50MB

4. **扫描版 PDF**
   - 仅支持基于文本的 PDF
   - 扫描版 PDF 需要先用 OCR 工具处理
