# 安装指南

## 快速开始

### 步骤 1: 准备图标文件

扩展需要图标文件才能正常加载。有两种方法：

#### 方法一：使用图标生成器（推荐）

1. 在浏览器中打开 `generate-icons.html`
2. 点击每个按钮下载对应的图标文件
3. 将下载的图标文件移动到 `icons/` 目录下

#### 方法二：手动创建图标

在 `icons/` 目录下创建以下文件：
- `icon16.png` (16x16 像素)
- `icon48.png` (48x48 像素)
- `icon128.png` (128x128 像素)

可以使用在线图标生成器或设计工具创建。

### 步骤 2: 加载扩展

1. 打开 Chrome 浏览器
2. 在地址栏输入 `chrome://extensions/` 并回车
3. 开启右上角的"开发者模式"开关
4. 点击"加载已解压的扩展程序"按钮
5. 选择 `book-splitter-extension` 文件夹
6. 扩展加载成功后，工具栏会显示扩展图标

### 步骤 3: 使用扩展

#### 方法一：通过扩展图标

1. 点击浏览器工具栏中的扩展图标
2. 按照界面提示选择文件并配置选项
3. 开始拆分并下载章节文件

#### 方法二：在 NotebookLM 页面

1. 访问 https://notebooklm.google.com
2. 页面右下角会出现"📚 拆分书籍"浮动按钮
3. 点击按钮后，按照提示点击扩展图标使用功能

## 故障排除

### 问题：扩展无法加载

**解决方案：**
- 确保所有必需的文件都在 `book-splitter-extension` 目录下
- 确保 `icons/` 目录下有图标文件
- 检查 `manifest.json` 文件是否存在且格式正确

### 问题：图标显示为灰色

**解决方案：**
- 确保 `icons/` 目录下有所有三个图标文件（16x16, 48x48, 128x128）
- 图标文件必须是 PNG 格式

### 问题：无法处理文件

**解决方案：**
- 确保网络连接正常（需要加载 CDN 库）
- 检查文件格式是否支持（PDF、EPUB、MOBI）
- 对于 MOBI 文件，如果无法解析，建议先转换为 EPUB 或 PDF

### 问题：下载失败

**解决方案：**
- 检查浏览器的下载权限设置
- 确保有足够的磁盘空间
- 如果同时下载多个文件失败，可以尝试分批下载
