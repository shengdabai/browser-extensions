# NotebookLM 章节拆分工具

一个强大的浏览器扩展，帮助您将书籍文件按章节拆分为独立文档，方便导入 NotebookLM 进行详细分析。

## ✨ 特性

- 📚 **支持多种格式**: PDF、EPUB、MOBI、AZW3
- 🎯 **智能章节检测**: 自动识别章节标题和结构
- 📄 **灵活拆分模式**: 按章节、长度、段落或智能拆分
- 🎨 **精美界面**: 渐变色设计，现代化用户体验
- ⚡ **快速处理**: 高效解析和拆分算法
- 💾 **批量下载**: 一键下载所有章节文件

## 📦 安装方法

### 从源码安装

1. 克隆或下载此仓库
2. 打开 Chrome 或 Edge 浏览器
3. 访问扩展管理页面：
   - Chrome: `chrome://extensions/`
   - Edge: `edge://extensions/`
4. 启用右上角的"开发者模式"
5. 点击"加载已解压的扩展程序"
6. 选择插件目录

## 🚀 使用方法

### 基本使用

1. **选择文件**
   - 点击浏览器工具栏中的扩展图标
   - 点击"选择书籍文件"按钮
   - 选择 PDF、EPUB 或 MOBI 文件

2. **配置选项**
   - ✅ 自动检测章节（推荐）
   - ✅ 包含章节编号

3. **查看章节预览**
   - 文件解析后自动显示检测到的章节列表
   - 每个章节显示标题和字符数

4. **开始拆分**
   - 点击"开始拆分"按钮
   - 等待处理完成
   - 章节文件会自动下载

### 在 NotebookLM 中使用

1. 访问 [NotebookLM](https://notebooklm.google.com)
2. 页面右下角会显示浮动按钮"📚 拆分书籍"
3. 点击后会提示您打开扩展
4. 按基本使用步骤操作即可

## 🔧 高级功能

### 章节拆分模式

插件支持多种拆分模式：

1. **自动模式** (推荐)
   - 自动检测章节标题
   - 保持原有章节结构
   - 适合有明确章节的书籍

2. **长度模式**
   - 按固定字符数拆分
   - 默认最大 50,000 字符
   - 适合长文本

3. **段落模式**
   - 按段落数量拆分
   - 默认每 50 个段落一章
   - 保持段落完整性

4. **智能模式**
   - 综合考虑句子和段落
   - 尽量在合适位置拆分
   - 平衡章节长度

### 章节识别规则

插件能够识别以下章节格式：

- 中文章节：`第一章`、`第1章`、`第零壹章`
- 数字章节：`第1章`、`Chapter 1`
- 罗马数字：`I.`、`II.`、`III.`
- Markdown 标题：`#`、`##`、`###`

## 📋 技术实现

### 架构

```
notebooklm-chapter-splitter/
├── manifest.json          # 扩展配置
├── popup.html/js/css      # 弹出窗口界面
├── fileParser.js          # 文件解析模块
├── chapterSplitter.js     # 章节拆分模块
├── background.js          # 后台服务
├── content.js             # NotebookLM 页面注入
└── icons/                 # 图标资源
```

### 核心技术

- **PDF.js**: PDF 文件解析
- **JSZip**: EPUB 文件解压
- **Chrome Extension API**: 扩展功能
- **DOMParser**: HTML/XML 解析

### 文件处理流程

```
文件选择 → 格式识别 → 内容提取 → 章节检测 → 内容拆分 → 批量下载
```

## 🎯 最佳实践

### PDF 文件

✅ **推荐**:
- 文本型 PDF（非扫描版）
- 有明确章节标题
- 文档结构清晰

❌ **不推荐**:
- 扫描版 PDF（需要先 OCR）
- 图片为主的书
- 没有章节结构的文档

### EPUB 文件

✅ **推荐**:
- 标准 EPUB 格式
- 包含完整目录
- 规范的 HTML 结构

### MOBI 文件

⚠️ **注意事项**:
- MOBI 格式复杂，解析有限
- 建议先用 Calibre 转换为 EPUB
- 或转换为 PDF 后使用

## 🔍 故障排除

### 问题：文件解析失败

**可能原因**:
- 文件格式不支持
- 文件已损坏
- 加密或受保护的文件

**解决方法**:
- 检查文件格式是否正确
- 尝试转换为其他格式
- 移除文件加密

### 问题：未检测到章节

**可能原因**:
- 章节格式不符合常见模式
- 书籍没有明确章节

**解决方法**:
- 使用"智能拆分"模式
- 手动调整章节识别规则
- 查看原始文本确认格式

### 问题：下载被阻止

**可能原因**:
- 浏览器下载设置
- 文件名包含特殊字符

**解决方法**:
- 允许多文件下载
- 检查下载权限设置
- 插件会自动清理文件名

## 📊 性能优化

### 处理速度

- 小文件 (< 10MB): 几秒钟
- 中等文件 (10-50MB): 10-30 秒
- 大文件 (> 50MB): 可能需要 1-2 分钟

### 内存使用

- 插件会在内存中加载完整文件
- 建议处理 < 100MB 的文件
- 超大文件可能导致浏览器卡顿
