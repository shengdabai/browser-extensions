## 🔧 常见安装问题

### 问题 1：找不到"开发者模式"

**解决方法：**
- 确保使用 Chrome 或 Edge 浏览器
- 开发者模式在页面右上角
- 需要向下滚动才能看到

### 问题 2：加载失败

**可能原因：**
- 文件夹路径不正确
- 权限问题
- 浏览器版本过旧

**解决方法：**
1. 检查文件夹路径是否正确
2. 确保选择的是文件夹而不是文件
3. 更新浏览器到最新版本
4. 重新加载扩展管理页面

### 问题 3：图标不显示

**解决方法：**
1. 刷新扩展管理页面
2. 点击扩展卡片上的"重新加载"按钮
3. 重启浏览器

### 问题 4：点击图标没反应

**解决方法：**
1. 检查 popup.html 是否存在
2. 打开浏览器控制台查看错误
3. 重新加载扩展
