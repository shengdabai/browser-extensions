# 安装指南

## 快速安装步骤

### Chrome浏览器

1. 打开Chrome浏览器
2. 在地址栏输入：`chrome://extensions/`
3. 打开右上角的 **"开发者模式"** 开关
4. 点击 **"加载已解压的扩展程序"** 按钮
5. 选择 `pinyin-extension` 文件夹
6. 安装完成！

### Edge浏览器

1. 打开Edge浏览器
2. 在地址栏输入：`edge://extensions/`
3. 打开左下角的 **"开发人员模式"** 开关
4. 点击 **"加载解压缩的扩展"** 按钮
5. 选择 `pinyin-extension` 文件夹
6. 安装完成！

## 图标文件说明

插件需要以下图标文件（可选，不影响功能）：
- `icon16.png` (16x16像素)
- `icon48.png` (48x48像素)
- `icon128.png` (128x128像素)

如果没有这些文件，插件仍然可以正常工作，只是工具栏图标会显示默认图标。

如果需要添加图标：
1. 准备三个PNG格式的图标文件（尺寸如上）
2. 将文件放在 `pinyin-extension` 文件夹中
3. 在扩展程序页面点击"重新加载"按钮

## 使用方法

1. 安装后，插件默认启用
2. 在任意网页的输入框中输入中文
3. 输入完成后，拼音会自动出现在中文后面
4. 点击浏览器工具栏的插件图标可以启用/禁用

## 测试

安装后可以这样测试：

1. 打开任意网页（如：百度、Google等）
2. 在搜索框中输入：`你好`
3. 输入完成后应该看到：`你好 (nǐ hǎo)`

## 常见问题

**Q: 拼音没有自动出现？**
A: 
- 检查插件是否已启用（点击插件图标查看）
- 刷新网页后重试
- 检查浏览器控制台是否有错误（F12打开开发者工具）

**Q: 拼音显示不准确？**
A: 
- 插件优先使用pinyin-pro库（需要网络连接）
- 如果网络无法访问CDN，会使用内置字典（覆盖常用字）
- 确保网络连接正常以获得最佳效果

**Q: 如何卸载？**
A: 
- 在扩展程序页面找到"中文拼音自动标注"
- 点击"移除"按钮即可

## 技术支持

如果遇到问题，请检查：
1. 浏览器版本（需要Chrome 88+ 或 Edge 88+）
2. 是否开启了开发者模式
3. 浏览器控制台是否有错误信息
