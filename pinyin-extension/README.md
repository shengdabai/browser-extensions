# 中文学习助手 - 拼音与翻译浏览器插件

这是一个专为外国人学习中文设计的Chrome/Edge浏览器插件。当您在浏览器或PDF中输入中文时，插件会自动在中文后面添加**带声调的拼音标注**和**英文翻译**，帮助您更好地理解和学习中文。

## ✨ 功能特点

### 核心功能
- ✅ **自动识别中文输入**：在浏览器中输入中文时自动检测
- ✅ **准确拼音转换**：使用专业的pinyin-pro库，支持所有常用汉字
- ✅ **完整声调标注**：显示准确的声调（ā á ǎ à）
- ✅ **英文翻译**：自动提供中文到英文的翻译
- ✅ **智能去重**：避免重复添加标注
- ✅ **灵活配置**：可以单独控制拼音和翻译的显示

### 支持场景
- 📝 普通输入框（input、textarea）
- ✏️ 可编辑元素（contenteditable）
- 📄 PDF注释和标注
- 🌐 所有网页和在线应用

### 显示格式
输入中文后，会自动显示：
```
中文 (pīn yīn) [English Translation]
```

**示例：**
- 输入：`你好`
- 显示：`你好 (nǐ hǎo) [Hello]`

- 输入：`学习中文`
- 显示：`学习中文 (xué xí zhōng wén) [Learn Chinese]`

## 📦 安装方法

### Chrome浏览器

1. 打开Chrome浏览器
2. 在地址栏输入 `chrome://extensions/` 并回车
3. 打开右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择 `pinyin-extension` 文件夹
6. 插件安装完成

### Edge浏览器

1. 打开Edge浏览器
2. 在地址栏输入 `edge://extensions/` 并回车
3. 打开左下角的"开发人员模式"
4. 点击"加载解压缩的扩展"
5. 选择 `pinyin-extension` 文件夹
6. 插件安装完成

## 🎯 使用方法

### 基本使用

1. 安装插件后，插件默认是启用状态
2. 在任意网页的输入框中输入中文
3. 输入完成后，拼音和英文翻译会自动出现在中文后面
4. 点击浏览器工具栏的插件图标可以配置选项

### 配置选项

点击插件图标，可以设置：

1. **启用插件**：开启/关闭整个插件功能
2. **显示拼音**：控制是否显示拼音标注
3. **显示英文翻译**：控制是否显示英文翻译

您可以根据学习需要，灵活组合这些选项：
- 只显示拼音：适合练习发音
- 只显示翻译：适合快速理解意思
- 同时显示：适合全面学习

## 🔧 技术说明

### 技术栈
- **拼音库**：优先使用pinyin-pro库（通过CDN加载），如果加载失败则使用内置的常用汉字字典
- **翻译服务**：使用Google Translate免费API接口
- **兼容性**：支持Chrome 88+ 和 Edge 88+
- **性能优化**：
  - 使用防抖机制，避免频繁触发转换
  - 翻译结果缓存，减少API调用
  - 智能去重，避免重复标注

### 工作原理
1. 监听页面上的输入事件
2. 检测新输入的中文字符
3. 异步获取拼音和翻译
4. 在光标位置插入标注
5. 保持光标位置，方便继续输入

## ⚙️ 配置说明

### 存储设置
插件使用Chrome的同步存储（chrome.storage.sync）保存设置，包括：
- `pinyinEnabled`: 是否启用插件（默认：true）
- `showPinyin`: 是否显示拼音（默认：true）
- `showTranslation`: 是否显示翻译（默认：true）

### 翻译缓存
插件会缓存翻译结果，避免重复翻译相同内容。缓存大小限制为500条。

## 📝 注意事项

1. **网络要求**：
   - 首次使用时，插件需要从CDN加载pinyin-pro库，可能需要几秒钟
   - 翻译功能需要网络连接，使用Google Translate API
   - 如果网络环境无法访问CDN，插件会使用内置的常用汉字字典

2. **标注位置**：
   - 标注会直接插入到文本中（输入框）或作为元素插入（可编辑区域）
   - 如果需要删除，可以手动删除括号内的内容

3. **性能考虑**：
   - 翻译API有使用频率限制，大量输入时可能会有延迟
   - 插件使用缓存机制优化性能

4. **隐私说明**：
   - 翻译功能会向Google Translate API发送文本
   - 拼音转换在本地完成，不会发送数据

## 🐛 常见问题

**Q: 拼音或翻译没有自动出现？**
A: 
- 检查插件是否已启用（点击插件图标查看）
- 检查是否开启了相应的显示选项（拼音/翻译）
- 刷新网页后重试
- 检查浏览器控制台是否有错误（F12打开开发者工具）
- 确认网络连接正常（翻译功能需要网络）

**Q: 翻译不准确？**
A: 
- 翻译使用Google Translate API，准确性取决于Google的翻译质量
- 对于专业术语或复杂句子，可能需要人工校对
- 可以尝试调整输入文本，使用更简单的表达

**Q: 拼音显示不准确？**
A: 
- 插件优先使用pinyin-pro库（需要网络连接）
- 如果网络无法访问CDN，会使用内置字典（覆盖常用字）
- 确保网络连接正常以获得最佳效果

**Q: 如何卸载？**
A: 
- 在扩展程序页面找到"中文学习助手 - 拼音与翻译"
- 点击"移除"按钮即可
