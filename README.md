# Browser Extensions

A collection of Chrome extensions for Chinese language learning and content processing.

## 📦 Extensions Overview

### 📚 Book Splitter Extension
A Chrome extension to split PDF, EPUB, MOBI books into individual chapters for detailed analysis in Google NotebookLM.

**Key Features:**
- Support for multiple book formats: PDF, EPUB, MOBI
- Intelligent chapter detection and recognition
- Multiple output formats: Plain text, Markdown, HTML
- Batch download of split chapters
- Automatic chapter numbering modes

### 🇨🇳 Pinyin Extension
Chinese learning assistant that adds pinyin annotations and English translations to help foreign learners.

**Key Features:**
- Automatic pinyin overlay for Chinese text
- English translation integration
- Works on any webpage
- Customizable display options
- Real-time text processing

### 📝 NotebookLM Chapter Splitter
Extension designed to split books into chapters specifically optimized for NotebookLM analysis.

**Key Features:**
- PDF, EPUB, MOBI format support
- NotebookLM integration
- Smart chapter detection
- One-click chapter processing
- Clean export formats

## 🚀 Installation

### Chrome Extension Installation

1. Open Chrome browser
2. Navigate to `chrome://extensions/`
3. Enable "Developer mode" (top right)
4. Click "Load unpacked"
5. Select the extension folder

## 📂 Directory Structure

```
browser-extensions/
├── book-splitter-extension/     # 书籍章节拆分工具
├── pinyin-extension/           # 中文学习助手 - 拼音与翻译
└── notebooklm-chapter-splitter/ # NotebookLM章节拆分工具
```

## 🎯 Usage Instructions

Each extension has its own detailed README. Please refer to the specific extension documentation for comprehensive usage guides.

### Quick Start
- Book Splitter: Upload book file → Configure options → Download chapters
- Pinyin Extension: Enable on any page with Chinese content → View annotations
- NotebookLM Splitter: Access NotebookLM → Use extension tools → Process books

## 🔧 Development

All extensions are built with Manifest V3 and include:
- Content scripts for web page integration
- Background service workers
- Popup interfaces
- Icon sets
- Comprehensive documentation

# 浏览器插件集合

用于中文学习和内容处理的 Chrome 插件合集。

## 📦 插件概览

### 📚 书籍章节拆分工具
Chrome 扩展程序，将 PDF、EPUB、MOBI 书籍拆分为独立章节，以便在 Google NotebookLM 中进行详细分析。

**主要功能:**
- 支持多种书籍格式：PDF、EPUB、MOBI
- 智能章节识别
- 多种输出格式：纯文本、Markdown、HTML
- 批量下载章节文件
- 多种章节编号模式

### 🇨🇳 中文学习助手 - 拼音与翻译
中文学习助手，自动添加拼音标注和英文翻译，帮助外国人学习中文。

**主要功能:**
- 中文文本拼音叠加
- 英文翻译集成
- 支持任何网页
- 可自定义显示选项
- 实时文本处理

### 📝 NotebookLM 章节拆分工具
专为 NotebookLM 分析设计的书籍章节拆分插件。

**主要功能:**
- PDF、EPUB、MOBI 格式支持
- NotebookLM 集成
- 智能章节检测
- 一键章节处理
- 清晰的导出格式

## 🚀 安装方法

### Chrome 插件安装

1. 打开 Chrome 浏览器
2. 访问 `chrome://extensions/`
3. 开启"开发者模式"（右上角）
4. 点击"加载已解压的扩展程序"
5. 选择对应的插件文件夹
